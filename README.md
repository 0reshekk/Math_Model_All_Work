# пример задачи
Условие транспортной задачи:
Холодильники с трех складов необходимо доставить четырем розничным магазинам сети EnergoSale. Количество холодильников на складах, требуемое количество поставок в магазины, а также стоимость перевозки данных товаров со складов в магазины известны и представлены в таблице 1.
Требуется составить опорный план перевозки холодильников и рассчитать стоимость грузоперевозки
Построение опорного плана выполнить методом, указанным в билете.
